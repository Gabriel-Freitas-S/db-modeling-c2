📋 Sobre Esta Atividade Prática

Esta é uma implementação completa da Atividade Prática C2 de Modelagem de Banco de Dados, apresentando 10 contextos práticos de sistemas de informação organizados por níveis de complexidade (Baixa → Média → Alta).

Cada contexto demonstra padrões específicos de modelagem de dados com diagramas MER interativos, scripts SQL DDL funcionais e documentação técnica detalhada.

📱 Role a página para explorar todos os contextos. Os diagramas são carregados automaticamente conforme você navega.

🎯 Sobre o Projeto

Implementação da Atividade Prática C2 de Modelagem de Banco de Dados com 10 contextos práticos organizados por complexidade.

Material educacional desenvolvido para demonstrar padrões de modelagem relacional com SQLite.

🚀 Tecnologias Utilizadas

  • Frontend: HTML5, CSS3, JavaScript ES6+
  • Diagramas: Mermaid.js (renderização MER)
  • Database: SQLite 3.x
  • Syntax Highlighting: Prism.js
  • Interface: Lazy Loading e UX otimizada
